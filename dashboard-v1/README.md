# Fitbit Personal Health Analytics & Dashboard V1

A comprehensive local Data Engineering and Data Science pipeline to analyze, visualize, and benchmark personal health data exported from Fitbit.

This project uses **Python** for the ETL process (cleaning, merging, statistics, metabolic calculations) and **Node.js/Chart.js** for an interactive local dashboard.

## ğŸš€ Features

- **Multi-Format Ingestion:** Handles mixed granulaties (intra-day JSONs vs daily CSVs) and timezones automatically.
- **Data Standardization:** Automatically converts weights from Lbs to **Kg** and standardizes dates.
- **Custom Metrics:**
  - **Readiness Score (Z-Score):** Combines Sleep Quality and Resting Heart Rate (RHR) deviation to recommend training intensity.
  - **Metabolic Analysis:** Calculates **BMR** (Mifflin-St Jeor) vs **Active Calories**.
  - **Intensity Index:** Measures caloric burn per minute of activity.
- **Interactive Dashboard:** A modern, dark-mode web UI (Vite + Chart.js) featuring:
  - **Date Range Picker:** Filter analysis by specific periods.
  - **Activity Zones:** Doughnut chart breakdown (Sedentary vs Active).
  - **Energy Composition:** Stacked bar chart (BMR vs Active Burn).
  - Dual-axis trends and Dynamic KPI cards.
- **Daily Briefing:** A CLI script for a quick text-based check-up.

## ğŸ“Š Analytics Logic

### 1. Readiness Score

Calculated using **Z-Scores** (Standard Deviations from your personal mean):

- `Readiness = (Sleep Z-Score) - (RHR Z-Score)`
- **Interpretation:** `> 1.0` (Peak), `0.0` (Average), `< -1.0` (Stressed).

### 2. Metabolic Metrics

- **BMR (Basal Metabolic Rate):** Calculated using the **Mifflin-St Jeor Equation** based on your config in `analyze.py`.
- **Active Calories:** `Total Calories - BMR`. Represents energy burned purely through movement.
- **Intensity Index:** `Active Calories / Total Active Minutes`. A measure of workout density/intensity (Kcal/min).
